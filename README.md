Customer Shopping Behavior Analysis
Technologies: Python, Pandas, NumPy, Matplotlib, Microsoft Excel, Power BI, MySQL
Collected, cleaned, and transformed customer shopping data using Python (Pandas, NumPy) and Excel. 
Performed Exploratory Data Analysis (EDA) to identify customer purchasing patterns, spending behavior, and product trends. 
Created interactive dashboards and visual reports in Power BI to track sales performance and customer insights. 
Used MySQL for data storage and querying, enabling efficient data retrieval and analysis.
